# Valentine's Website Project

A romantic, interactive 3-page Valentine's Day website.

## Features

1.  **Landing Page**: "Will you be my Valentine?" with interactive buttons.
2.  **Letter Page**: A heartfelt handwritten-style letter.
3.  **Memories Gallery**: A photo gallery with a confetti celebration effect.

## How to Run Locally

1.  Clone the repository.
2.  Open `index.html` in your browser.

## Deployment

This website is designed to be hosted on GitHub Pages.

1.  Push the code to a GitHub repository.
2.  Go to repository **Settings** > **Pages**.
3.  Select the **main** branch as the source.
4.  Save. Your site will be live at `https://<username>.github.io/<repo-name>/`.
